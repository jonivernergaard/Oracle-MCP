# Migration Visualizer Frontend

This is a React + Vite application for the Migration Visualizer.

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the backend server (in the root directory):
    ```bash
    python web_server.py
    ```

3.  Start the frontend development server:
    ```bash
    npm run dev
    ```

## Features

-   **React + Vite**: Modern, fast frontend tooling.
-   **Tailwind CSS**: Utility-first CSS framework.
-   **Tabulator**: Advanced data grid for displaying migration results.
-   **Document Viewer**: Enhanced modal for viewing source documents with highlighting support for both Markdown and Source views.
-   **Highlighting Fix**: Improved highlighting logic for Markdown preview mode.
